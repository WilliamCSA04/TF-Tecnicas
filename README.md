# TF-Tecnicas

### Build Tools
 - Gradle

### Dependências
 - Google Guice
 - JUnit
 - PostgreSQL JDBC 9.4
 
### Build
 
**Windows:**
  - gradlew.bat
**Linux:**
 - ./gradlew
 
Para ver as possibilidades de tasks disponíveis no gradle, execute a task `tasks`:

Windows: `gradlew.bat tasks`

Linux: `./gradlew tasks`
